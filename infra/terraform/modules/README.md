# Terraform modules

This directory contains reusable Terraform modules for the Troubleshoot Stack MVP.

## Modules
- vpc
- ecs_service
- lambda_service (optional)
- apigw
- dynamodb
- s3
- storage
- observability
- iam

Each module includes `variables.tf`, `outputs.tf`, and a module-level `README.md` that documents inputs and outputs.
