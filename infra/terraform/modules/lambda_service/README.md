# lambda_service module

Creates a Lambda function and supporting resources.

## Inputs
- `function_name`
- `handler`
- `runtime`
- `memory_mb`
- `timeout_s`
- `env_vars`
- `role_arn`

## Outputs
- `function_arn`
- `invoke_arn`

## Notes
This module is currently a scaffold; wire resources in later steps.
